Pipeline Overview

1

Qwen3-VL Texture Analysis

Each image is paired with its colored segmentation mask and sent to Qwen3-VL-8B. The model identifies adjacent texture transitions and outputs JSON with 5–9 word discriminative descriptions and mask color assignments for each texture region.

2

Binary Mask Extraction

RGB color matching extracts binary masks from the segmentation. Connected-component filtering ensures only physically adjacent regions are paired. Duplicate transitions are removed by mask IoU and text similarity.

3

Boundary Crop Extraction

The boundary between textures is skeletonized. Anchor points are sampled along it, and a binary search finds the maximal crop at each anchor across multiple aspect ratios. NMS removes overlapping crops. Third-class pixels are assigned to the nearest texture by color distance.

4

Super-Resolution Refinement

All crops are upscaled 2× using Real-ESRGAN x2plus, then clipped to a maximum of 512px per side. This brings the average crop from 103px to 206px.

5

SDF Mask Smoothing

Signed distance fields are computed for each mask. A Gaussian-smoothed SDF is thresholded at zero to produce complementary masks with smooth polynomial-like boundaries — zero gaps, zero overlap, every pixel assigned exactly once.
